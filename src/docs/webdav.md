# WebDAV 介绍

DocBase 自带 WebDAV 服务，你可以<mark>通过 WebDAV 增删改查知识库</mark>文件，DocBase 会自动<mark>同步更改</mark>到知识引擎中。

访问`/dav`路径即可访问 WebDAV 服务

例如：

```
http://your.domain/dav
```

webdav 用户名: docbase

webdav 密码: MEILI_MASTER_KEY 的值
